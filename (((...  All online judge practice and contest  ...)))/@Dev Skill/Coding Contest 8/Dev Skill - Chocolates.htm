Easy Math > Basic Math

Your father give you n chocholates.you go to the school and you will now play with this chocolates........... you decorate them in a good way like below

                             c
                            c c
                           c c c
                          c c c c
                         c c c c c
                        c c c c c c

Now you have to calculate the number of lines l formed by your chocolates.if you cant complete a line then that line will be removed and eat the remaining chocolates.

Input:

The first line of input will be an integer t number of test case. next t lines contain an integer N the number of chocolates.

Output:

Print the number of lines l on the first line and then the number of chocolates you eat on the next line.

Constrains

1<=t<=100 1<=N<=10^18

Sample Input

2
10
22

Sample Output

4
0
6
1
